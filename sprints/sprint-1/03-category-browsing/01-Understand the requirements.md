# Browse Products by Category

## Product Overview

### User Story

As a visitor, I want to browse products within a specific category, so that I can find products of a particular type.

---

## Acceptance Criteria

### AC1 – Category page is displayed

**Requirement**

Given I click on a category name  
Then a page with products belonging to that category is displayed.

**Understanding**

The user should be able to navigate to a category page by selecting a category.

---

### AC2 – Category title

**Requirement**

Given the category page is displayed  
Then the category name is shown as the page title.

**Understanding**

The category page should clearly identify which category the user is currently viewing.

---

### AC3 – Products from selected category

**Requirement**

Given the category page is displayed  
Then only products belonging to the selected category are shown.

**Understanding**

The category page should display only products that belong to the selected category.

**Open Questions / Assumptions**

- How is it determined that a product belongs to the selected category?
- Is the order of the displayed products important?
